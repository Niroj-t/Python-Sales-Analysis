# Diwali Sales Analysis

## Overview
This project performs exploratory data analysis (EDA) on Diwali sales data using Python and Jupyter Notebook. The analysis covers data cleaning, summary statistics, and visualizations that reveal customer demographics and sales trends.

## Sales Report 
https://docs.google.com/document/d/1oKFu58uLW7kUeN2x2d9iEj1waEmR0TP2/edit#heading=h.cbrtxj7jijf8


## Files
- `Sales_Analysis.ipynb` - Jupyter Notebook containing the data import, cleaning, and visualization steps.
- `Sales Data.csv` - Dataset used for the analysis.

## Key Analysis Areas
- Data cleaning and preprocessing
- Sales distribution by gender
- Sales trends by age group
- Top performing states by order count and total sales amount

## Requirements
- Python 3.x
- Jupyter Notebook
- pandas
- numpy
- matplotlib
- seaborn

## How to Run
1. Open `Sales_Analysis.ipynb` in Jupyter Notebook or JupyterLab.
2. Ensure `Sales Data.csv` is present in the same directory.
3. Run the notebook cells sequentially to reproduce the analysis and visualizations.

## Notes
- The notebook uses `unicode_escape` encoding when reading the CSV file.
- Null values are removed and irrelevant columns are dropped during preprocessing.
